# Find Your Hat

A simple game project where you must find your hat on a grid while avoiding obstacles.

## How to Play

- Use arrow keys to move your character.
- Avoid falling into holes.
- Find your hat to win the game.

## Features

- Randomly generated maps
- Multiple difficulty levels
- Replayable gameplay

## Installation

1. Clone the repository:
    ```
    git clone https://github.com/yourusername/find-your-hat.git
    ```
2. Navigate to the project directory:
    ```
    cd find-your-hat
    ```
3. Run the game according to the instructions in the `main` file.
